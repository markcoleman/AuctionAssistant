## How to Use These Templates

### Understanding Template Structure

Each template demonstrates different approaches to marketplace listings:

1. **Tone Variations**
   - Professional: Formal, trustworthy, detailed
   - Casual: Friendly, conversational, relatable
   - Enthusiastic: Energetic, exciting, passionate
   - Luxury: Sophisticated, premium, exclusive
   - Bargain: Value-focused, deal-oriented, cost-conscious

2. **Style Variations**
   - Feature-Focused: Emphasizes specifications and features
   - Benefit-Focused: Highlights how product helps buyer
   - Story-Based: Tells a narrative about the product
   - Concise: Brief and to-the-point
   - Detailed: Comprehensive and thorough

### A/B Testing

Each template includes A/B testing variants that demonstrate different approaches:
- **Variant A**: One specific angle or emphasis
- **Variant B**: Alternative angle or different feature focus
- **Variant C**: Another unique approach

Use these to understand how different messaging can appeal to different buyer types.

## Using the Post Generation Service

### Basic Usage

```typescript
import { createPostGenerationService } from './src/services/postGenerationService';
import { MarketplaceTone, DescriptionStyle } from './src/prompts/marketplacePrompts';

// Initialize the service
const postService = createPostGenerationService();

// Generate a post from product analysis
const result = await postService.generatePost(productAnalysis, {
  tone: MarketplaceTone.PROFESSIONAL,
  style: DescriptionStyle.FEATURE_FOCUSED,
  includeEmojis: true,
  addCTA: true,
  generateVariants: true,
  variantCount: 2
});

if (result.success) {
  console.log('Title:', result.primaryPost.title);
  console.log('Description:', result.primaryPost.description);
  console.log('Selling Points:', result.primaryPost.sellingPoints);
  
  // A/B testing variants
  result.variants?.forEach(variant => {
    console.log(`\nVariant (${variant.differentiatingFactor}):`);
    console.log('Title:', variant.title);
  });
}
```

## Best Practices

### 1. Choose the Right Tone
- **Professional**: Business items, high-value electronics
- **Casual**: Everyday items, clothing, books
- **Enthusiastic**: Trending items, collectibles, games
- **Luxury**: Designer items, jewelry, premium goods
- **Bargain**: Used items, clearance, quick sales

### 2. Match Style to Product
- **Feature-Focused**: Tech products, appliances
- **Benefit-Focused**: Health/fitness, tools, solutions
- **Story-Based**: Vintage items, handmade goods
- **Concise**: Quick turnover items, simple products
- **Detailed**: Complex items, high-value goods

### 3. A/B Testing Strategy
- Test different tones (professional vs. casual)
- Test different opening hooks
- Test feature emphasis vs. benefit emphasis
- Test different CTAs (call-to-actions)
- Track which variants get more engagement

### 4. Emoji Usage
- Use 2-4 emojis per post (not too many!)
- Place strategically (beginning, key points, end)
- Match emojis to product category
- Avoid overusing the same emojis

### 5. Title Optimization
- Keep within 50-80 characters
- Include brand name and key feature
- Mention condition for used items
- Avoid ALL CAPS and excessive punctuation
- Use power words (Amazing, Perfect, Premium)

### 6. Description Structure
- Start with an attention-grabbing hook
- Use bullet points for key features
- Include dimensions/specifications
- Be honest about condition and flaws
- End with a strong call-to-action
- Use paragraph breaks for readability

## Integration Example

```typescript
// Complete workflow from image to marketplace post
import { createAIVisionService } from './src/services/aiVisionService';
import { createPostGenerationService } from './src/services/postGenerationService';

async function createMarketplaceListing(imagePath: string) {
  // Step 1: Analyze product image
  const visionService = createAIVisionService();
  const analysisResult = await visionService.analyzeProduct(imagePath);
  
  if (!analysisResult.success || !analysisResult.data) {
    throw new Error('Failed to analyze product');
  }
  
  // Step 2: Generate marketplace post
  const postService = createPostGenerationService();
  const postResult = await postService.generatePost(analysisResult.data, {
    tone: MarketplaceTone.PROFESSIONAL,
    style: DescriptionStyle.FEATURE_FOCUSED,
    includeEmojis: true,
    generateVariants: true
  });
  
  if (!postResult.success || !postResult.primaryPost) {
    throw new Error('Failed to generate post');
  }
  
  // Step 3: Return complete listing
  return {
    title: postResult.primaryPost.title,
    description: postResult.primaryPost.description,
    sellingPoints: postResult.primaryPost.sellingPoints,
    variants: postResult.variants,
    metadata: {
      category: analysisResult.data.category.primary,
      condition: analysisResult.data.condition,
      tokensUsed: (analysisResult.tokensUsed || 0) + (postResult.totalTokensUsed || 0)
    }
  };
}
```
